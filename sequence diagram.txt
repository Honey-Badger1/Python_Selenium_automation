Key Understanding (Very Important)
🔹 Who calls whom?
Caller	Calls
Jenkins	CSRegistration.py
CSRegistration.py	Django API
Django View	Lib class
Lib class	SSH + OS commands
🔹 Where is main()?

❌ There is NO main()
✔ Django view is the entry point






Final Mental Model (Remember This)

TAF v2 = MVC + API + Automation

Model  → context.json + input
View   → Django REST View
Logic  → Lib file (SSH / Selenium)
